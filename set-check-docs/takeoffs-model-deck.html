01

The idea

Turning a drawing set into an order is manual and slow. Someone counts every window and door off the sheets, matches each size to what the manufacturer actually sells, and builds the quantity list by hand — the same work redone by the architect, the developer, and the supplier, and a miscount means a re-order or a delay. We think a tool can do that read in one pass. Here's how we picture it.

How it's done today

Read the schedules by eye. Tally sizes. Flip through the 100 and 400 Series catalogs to find the matching unit. Build the count in a spreadsheet — and hope nothing was missed.

The idea

Point it at our set and the brochures. It returns a ready-to-order chart — window units matched to Andersen catalog numbers with quantities, and doors counted by size — in one pass.

How far it could go — and what's hard

The window & door order chart is the first piece — but it's one read of our drawing set, and that single read can do far more. It can total the rest of what a project buys, and on the same pass check that what you bought matches our drawings and passes code. Here's the whole of it in one list — every piece with how big a build it is, ordered from what's essentially ready to the genuinely hard, new work.

Two of these we've already run — the order chart above and the window energy check. Everything in the top group builds on that same reading and is close; the area & linear-foot takeoffs at the bottom are the genuinely hard, new part — a real measurement engine, not a table lookup.
